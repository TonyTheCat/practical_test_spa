# front

## Project setup

```
1. Install npm modules `npm install`
2. Be sure of versions: `node version v14.19.0` and `npm version 8.5.0`
3. Set `ENV VARS`:
.env.example:
VUE_APP_CREATE_CLIENT=http://localhost:3000/client/create
VUE_APP_READ_CLIENTS=http://localhost:3000/client/get_all
VUE_APP_UPDATE_CLIENT=http://localhost:3000/client/update
VUE_APP_DELETE_CLIENT=http://localhost:3000/client/delete

VUE_APP_CREATE_PROVIDER=http://localhost:3000/provider/create
VUE_APP_READ_PROVIDERS=http://localhost:3000/provider/get_all
VUE_APP_UPDATE_PROVIDER=http://localhost:3000/provider/update
VUE_APP_DELETE_PROVIDER=http://localhost:3000/provider/delete
```
